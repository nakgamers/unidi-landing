# 🚀 Getting Started

### Clone Repository

```
git clone https://github.com/nakgamers/unidi-landing.git
```

### Masuk ke Folder Project

```
cd unidi-landing
```

### Jalankan

Buka file berikut di browser

```
index.html
```

atau gunakan **Live Server di VSCode**
